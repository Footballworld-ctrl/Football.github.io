CSS3 선택자

6가지 핵심 선택자를 코드와 함께 직접 확인하세요

01

Hover 선택자

:hover

요소에 마우스를 올렸을 때 스타일을 적용합니다. 클릭 없이도 시각적 피드백을 줄 수 있어 버튼, 링크, 카드 등에 광범위하게 사용됩니다.

.btn:hover {
  background: #e8ff47;
  color: #000;
  transform: translateY(-2px);
}
▶ 직접 해보기
02

N번째 자식 선택자

:nth-child()

부모 요소 내에서 순번에 따라 특정 자식을 선택합니다. odd / even 또는 수식 3n+1 형태로 패턴 선택이 가능합니다.

li:nth-child(odd) {
  background: rgba(232,255,71,0.15);
  border-color: #e8ff47;
  color: #e8ff47;
}

03

가상 요소 선택자

::before / ::after

HTML을 수정하지 않고도 요소의 앞(before) 또는 뒤(after)에 콘텐츠를 삽입합니다. 장식적 요소나 아이콘을 CSS만으로 구현할 때 강력합니다.

.quote::before {
  content: '\201C'; /* " */
  font-size: 2.2rem;
  color: #47c8ff;
}
▶ 따옴표 자동 삽입
가상 요소로 HTML 없이 콘텐츠를 추가할 수 있습니다.
04

부정 선택자

:not()

괄호 안의 조건에 해당하지 않는 요소만 선택합니다. 특정 클래스를 제외한 나머지 요소에 스타일을 적용할 때 매우 유용합니다.

span:not(.active) {
  background: rgba(255,255,255,0.04);
  color: var(--muted);
}
▶ .active 제외 흐리게 표시

05

속성 선택자

[attr] / [attr="val"]

HTML 속성의 유무 또는 값에 따라 요소를 선택합니다. [required], [type="email"] 등 폼 요소 스타일링에 특히 강력합니다.

input[required] {
  border-color: #ff6b6b;
}
input[disabled] {
  opacity: 0.35;
}
▶ 속성별 스타일 차이 확인

빨간 테두리 = [required] 속성 적용

06

포커스-내부 선택자

:focus-within

자식 요소 중 하나라도 포커스를 받으면 부모 요소에 스타일을 적용합니다. 폼 그룹 전체를 활성화 상태로 시각화할 때 JavaScript 없이 구현 가능합니다.

.form:focus-within {
  border-color: #e8ff47;
  background: rgba(232,255,71,0.04);
}
▶ 입력창 클릭 시 박스 전체 활성화

※ 입력 필드에 포커스를 주면 부모 박스가 강조됩니다
